# 05. Start Unreal 2026 - Week 1

**작성일 : 2026-07-13**

---

# 1. 실습 목적

이번 실습에서는 '시작해요 언리얼 2026' 강의를 따라 하면서 언리얼 엔진의 기본적인 사용 방법을 익히는 것을 목표로 하였다.

또한 프로젝트를 생성하고 기본적인 오브젝트를 배치하면서 언리얼 엔진의 작업 방식에 익숙해지고자 하였다.

---

# 2. 실습 내용

이번 실습에서는 다음 내용을 진행하였다.

- Starter Pack 프로젝트 실행
- 레벨 열기 및 저장
- Static Mesh 배치
- Move, Rotate, Scale 사용
- Alt 키를 이용한 오브젝트 복제
- Outliner 사용
- Sequencer 생성
- Cine Camera Actor 배치
- 카메라 애니메이션 만들기

---

# 3. 그래픽스 파이프라인 이해

교수님께서 말씀해 주신 것처럼 이번 실습에서는 단순히 따라 만드는 것이 아니라 그래픽 요소들이 어떻게 연결되는지 이해하려고 노력하였다.

특히 다음 요소들의 역할을 확인하였다.

- Mesh
- Material
- Lighting
- Camera
- Sequencer

아직 모든 기능을 이해한 것은 아니지만 앞으로 하나씩 공부할 예정이다.

---

# 4. 실습 중 어려웠던 점

실습을 진행하면서 여러 가지 어려움이 있었다.

- Starter Pack 프로젝트와 Blank 프로젝트를 혼동하였다.
- Alt 키를 이용한 복제가 잘되지 않았다.
- Sequencer에서 오브젝트 크기가 변경되는 이유를 이해하지 못하였다.
- 레벨을 저장하지 않아 작업한 내용을 잃어버렸다.

문제가 발생할 때마다 원인을 찾아 해결하면서 언리얼 엔진의 사용 방법을 조금씩 익힐 수 있었다.

---

# 5. 느낀 점

이번 실습은 단순히 영상을 따라 하는 것보다 언리얼 엔진의 기본적인 구조를 이해하는 데 의미가 있었다.

처음에는 모든 것이 낯설었지만 직접 오브젝트를 배치하고 카메라를 움직여 보면서 언리얼 엔진의 작업 방식을 조금씩 이해할 수 있었다.

앞으로는 Material, Lighting, Water System 등 그래픽스와 관련된 기능도 계속 공부하면서 언리얼 엔진에 익숙해지고 싶다.
