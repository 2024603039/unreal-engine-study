# 시작해요 언리얼 2026 - 1주차 실습 정리

## 1. 학습 주제
블록아웃 기법과 시퀀서를 활용하여 기본 레벨과 시네마틱 장면을 구성하였다.

## 2. 주요 학습 내용
- Ignite Starter Pack 프로젝트 실행
- 레벨 저장 및 불러오기
- Static Mesh 배치
- 이동, 회전, 크기 조절
- Alt 드래그를 이용한 액터 복제
- Outliner 폴더를 활용한 액터 정리
- Cine Camera Actor 사용
- Sequencer 타임라인과 키프레임
- Camera Cut을 활용한 카메라 연출
- Landscape와 환경 오브젝트 배치

## 3. 실습 과정
강의에서 제공한 Ignite Starter Pack을 다운로드한 후 `IgniteUnreal.uproject`를 실행하였다. 바닥, 원기둥, 바위, 캐릭터 등의 액터를 배치하고 Transform을 조절하였다. 이후 Sequencer에서 Cine Camera Actor와 Camera Cut을 사용하여 카메라 시점을 구성하였다.

## 4. 발생한 문제와 해결
- 기존 빈 프로젝트와 강의 프로젝트를 혼동함  
  → 강의용 `IgniteUnreal.uproject`를 직접 실행하여 해결함.
- 레벨 저장 시 계속 이름을 묻는 창이 나타남  
  → 무제 레벨을 처음 저장한 뒤부터 일반 저장이 가능해짐.
- 시퀀서 카메라 시점에서 원기둥 크기가 원래대로 돌아감  
  → Transform의 Scale 키프레임이 값을 덮어쓰는 문제임을 확인함.
- Alt 드래그 복제가 되지 않음  
  → 이동 도구 상태에서 축을 잡고 Alt를 누른 채 드래그해야 함을 확인함.
- 배치한 액터가 많아 관리가 어려움  
  → Outliner에서 여러 액터를 선택한 뒤 폴더로 묶어 정리함.

## 5. 실습 결과
기본 도형과 환경 에셋을 활용해 블록아웃 장면을 구성하고, Sequencer를 이용한 카메라 연출까지 완료하였다.

## 6. 느낀 점
언리얼 엔진에서는 액터 배치뿐 아니라 Outliner 정리, 레벨 관리, 시퀀서 키프레임 관리가 중요하다는 점을 알게 되었다. 특히 시퀀서가 액터의 Transform 값을 제어할 수 있어 일반 뷰포트에서 수정한 값이 다시 변경될 수 있다는 점을 실습을 통해 이해하였다.
