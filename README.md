# Unreal Engine 5.8 MCP Study

> Unreal Engine 5.8의 Model Context Protocol(MCP)을 활용하여 AI Agent Team 기반의 언리얼 엔진 개발 환경을 학습한 저장소입니다.

---

# Project Overview

본 저장소는 Unreal Engine 5.8에서 제공하는 MCP(Model Context Protocol)를 활용하여 AI 에이전트(Codex)를 언리얼 엔진과 연동하고, Agent Team 기반 개발 방식을 학습한 내용을 정리하기 위해 제작되었습니다.

본 프로젝트에서는 단순히 AI에게 명령을 수행하도록 하는 것이 아니라 여러 Agent가 각각 역할을 분담하여 하나의 프로젝트를 수행하는 방식을 실습하였습니다.

# Agent Team

본 프로젝트에서는 다음과 같은 AI Agent Team을 구성하여 작업을 수행하였습니다.

- Project Manager
- Research Agent
- Level Designer Agent
- Technical Artist Agent
- QA Agent

각 Agent는 서로 다른 역할을 수행하며 Project Manager가 전체 작업을 관리합니다.
